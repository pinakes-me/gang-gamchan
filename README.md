# 별이 떨어진 자리 — 강감찬 탐험 지도

**🔗 살펴보기: https://pinakes-me.github.io/gang-gamchan/**

관악구립도서관 강감찬자료실의 도서 목록 239건을,
서로 연결된 지식 그래프와 관악의 실제 장소 위에서 탐험할 수 있게 만든
개인 연구 프로토타입입니다.

낙성대(落星垈)는 "큰 별이 떨어진 터"라는 뜻입니다.
천 년 전 그 별에서 태어났다는 강감찬 장군의 이야기가,
지금도 관악의 도서관 곳곳에 책으로 흩어져 있습니다.
이 지도는 그 책들과 장소들을 다시 잇습니다.

## 이렇게 탐험해보세요
- 🗺️ **지도의 별**을 누르면 — 낙성대공원, 생가터, 강감찬전시관 같은 실제 장소와
  얽힌 이야기·자료가 나타납니다
- 📚 **지도의 도서관**을 누르면 — 그 도서관에서 지금 빌릴 수 있는 강감찬 책이 나옵니다
- ┅ **노란 점선(강감찬 10리길)**을 누르면 — 걷기 전에 읽어볼 책들을 모아 보여줍니다
- 🕸️ **오른쪽 그래프의 개념**을 누르면 — 귀주대첩, 고려청자, 낙성대…
  주제로 연결된 자료들을 탐험할 수 있습니다

## 데이터와 방법
- 원천: 관악구립도서관 강감찬자료실 도서목록 (공개 서지정보) 전수 239건
- 파이프라인: CSV → SKOS 통제어휘(71개 개념) → RDF/Turtle(2,586 트리플) → 시각화
- 이 저장소에는 데이터(`data/`), 어휘(`vocab/`), 변환 스크립트(`scripts/`),
  생성된 RDF(`output/`)가 모두 공개되어 있어 누구나 재현할 수 있습니다
- 장소 좌표: Google Places, 서울도서관, 관악구 동주민센터 작은도서관 현황

## 만든 사람
사서 · 디지털인문학 연구자 **@pinakes_me**
공공도서관 향토자료실의 Linked Open Data 적용 가능성을 탐구하는
개인 연구의 일부입니다. 본 프로젝트는 공개된 서지정보를 활용한
연구 목적의 프로토타입이며, 관악구립도서관의 공식 서비스가 아닙니다.

자료의 실제 대출 가능 여부는 관악구통합도서관(lib.gwanak.go.kr)에서
확인해 주세요.
